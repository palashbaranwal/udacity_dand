Test a Perceptual Phenomenon - Stroop Effect¶

Background Information¶

In a Stroop task, participants are presented with a list of words, with each word displayed in a color of ink. The participant’s task is to say out loud the color of the ink in which the word is printed. The task has two conditions: a congruent words condition, and an incongruent words condition.

  1. In the congruent words condition, the words being displayed are color words whose names match the colors in which they are printed: for example RED, BLUE.
  2. In the incongruent words condition, the words displayed are color words whose names do not match the colors in which they are printed: for example PURPLE, ORANGE.

In each case, we measure the time it takes to name the ink colors in equally-sized lists. Each participant will go through and record a time from each condition.

Q2: What is an appropriate set of hypotheses for this task? What kind of statistical test do you expect to perform? Justify your choices.

Let us consider μi as reaction time under incongruent conditions, and μc as reaction time under congruent conditions.

The Null Hypothesis Ho is described as, when there is no observable difference in the average response time for identifying congruent words(μc) when compared to viewing the incongruent words(μi).

μi - μc = 0

The Alternative Hypothesis Ha is described as, when there is a observable difference in the average response time for identifying congruent words(μc) when compared to viewing the incongruent words(μi).

μi - μc ≠ 0

Here μ conveys the population mean.

We can consider bootstrapping, where we can use the sample function to extrapolate the dataset as a whole, and simulate the distribution with the size of the boostrapped differences.

Conclusion¶

Based on our p-value calculation and the visualization, we can reject the null hypothesis.

The p-value is zero which is less than significant level, indicating that there are not any values in the statistics which could match with the values generated under the null hypothesis. According to our observations, we can conclude that reaction time of participants is higher under incongruent conditions.
